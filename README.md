# Проект NetNexus
---------------------------
## NetNexus
Это инструмент для ускорения и автоматизации управления компьютерами. Управление компьютерами осуществляется тремя способами

- скрипты для групп и одиночных случаев (на выбор)
- удалённый терминал (только к одному компьютеру)
- удалённый экран как в anydesk или veyon

### Инструменетарий
- возможность отслеживать ресурсы компьютеры
- установленные программы
- скрипты для автоматизации
- удалённый монитор
- управление переферией
- введение базы данных для администратора(выборочное сохранение данных)
- встроенная возможность писать скрипты с проверкой на выброном компьютере
- возможность смотреть камеры (для администратора)
- создание групп в списке компьютеров для структурирования компьютеров (для администратора)
- назначение главного компьютера для группы и его подчинённых в группе(для администратора)

Каждый инструмен доступен в приложении NetNexus

### Приложение NetNexus
Предостовляет доступ к обширному списку возможностей и инструментов оно будет написано на Tauri с использованием таких инструментов как

- React + Typescript для фронтенда
- Rust для бэкэнда
- специальный драйвер для инструментария
